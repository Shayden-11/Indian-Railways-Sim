# Indian Railways Simulator

An immersive, web-based 2D train simulator built with Phaser 3. Command high-horsepower locomotives like the WAP-7 and WAP-5 across iconic Indian corridors with realistic physics and operational logic.

## 🚀 (https://github.com/Shayden-11/Indian-Rwailways-Sim.git)

---

## 🚆 Key Features

### 🕹️ Advanced Operations
* **Realistic Signalling:** Features a functional Absolute Block Signalling system (Red/Yellow/Green) with cascade logic.
* **Pneumatic Braking:** Authentic simulation of Brake Pipe (BP) and Brake Cylinder (BC) pressure dynamics.
* **Dynamic Environment:** Real-time day/night cycle based on system clock and procedural ghat scenery.

### 🔧 Customization & Fleet
* **Consist Builder:** A robust UI to build custom rakes using LHB (AC/Executive), Vistadome, Humsafar, and Pantry coaches.
* **Locomotive Choice:** Switch between the high-acceleration WAP-5 and the powerhouse WAP-7.
* **Iconic Liveries:** Accurate sprites for Rajdhani, Shatabdi, and Humsafar Express sets.

### 🗺️ Routes
* **Mumbai Central ⇌ New Delhi:** The fast-paced Rajdhani corridor.
* **Mumbai CST ⇌ Pune:** Featuring the legendary Bhor Ghat section with 45 km/h speed restrictions.
* **Coimbatore ⇌ Chennai Central:** A high-speed dash across the Southern Railway main line.

---

## 📻 Immersion
* **Guard Communication:** Radio chatter triggers for departure clearance and signal-clear calls.
* **Overspeed Protection:** Visual HUD warnings when exceeding section speed limits.
* **Station Progress:** Automated distance tracking and real-world scaled station sequencing.

---

## 🛠️ Built With
* **Engine:** Phaser 3 (Arcade Physics)
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Audio:** High-fidelity spatial horn and rail sounds

---

## 📜 How to Play
1. Open the https://github.com/Shayden-11/Indian-Rwailways-Sim.git.
2. Select your route and train set.
3. Use **Arrow Keys (Up/Down)** for Throttle and Brakes.
4. Press **'E'** for Emergency Brakes.
5. Use **Right-Click + Drag** to pan the camera across the train.
